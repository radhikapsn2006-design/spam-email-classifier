# Spam Email Classifier using NLP

This project classifies messages as spam or ham using Natural Language Processing and Machine Learning.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Algorithm

## 📌 Features
- Text preprocessing
- TF-IDF vectorization
- Spam classification
- Accuracy evaluation

## 🌍 Real-World Applications
- Gmail spam filtering
- SMS spam detection
- Fraud message detection
